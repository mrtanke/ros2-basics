## ros2 basics

Learning and re-implementation code while studying [**ROS 2 Robot Development: From Beginner to Practice**](https://www.bilibili.com/video/BV1GW42197Ck/?share_source=copy_web&vd_source=3972bd8dedb6595ff67ca476fa44cf83).

### Structure

- `chapt1/` - CMake basics and hello world programs
- `chapt2/` - ROS 2 packages and basic nodes (C++ and Python)
- `chapt3/` - Topic communication (publishers and subscribers)
- `chapt4/` - Service communication and custom interfaces
- `chapt5/` - Transform (TF) system and RViz
- `chapt6/` - Robot description (URDF) and visualization
- `chapt7/` - Autonomous inspection robot with Navigation 2
- `chapt8/` - Pluginlib and custom controllers/planners
